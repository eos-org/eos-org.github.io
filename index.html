Eos - Dawn is Coming

PHP Extensions for PHP7

Welcome to Eos

The goal of this organization is to provide useful, well documented, well designed extensions for PHP7. It's also designed to be a place to learn about PHP extension writing "the right way" since best practices are generally copy and paste 10 year old code nastiness.

Get Involved

Starting the beginning of this year auroraeosrose (and hopefully others) will be running a mentoring program for people wishing to adopt, create, or maintain PHP extensions. If you'd like to get involved send a message to auroraeosrose@gmail.com with "2016 Extension Mentoring" in the subject.
